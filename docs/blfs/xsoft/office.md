# Office Programs {#office-programs}

<div class="chapter" lang="en">
This chapter is a collection of programs that are useful for viewing or editing office documents. Some specialise in doing one thing (such as word processing or manipulating a spreadsheet). <span class="application">LibreOffice</span> is a suite of programs that can manipulate many different formats including powerpoint presentations.
