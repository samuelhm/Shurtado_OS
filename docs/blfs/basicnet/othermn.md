# Other Mail and News Programs {#other-mail-and-news-programs}

<div class="sect1" lang="en">
<a class="xref" href="../xsoft/balsa.md" title="Balsa-2.6.5">Balsa-2.6.5</a> is a GTK2 based mail client.

<a class="xref" href="../xsoft/seamonkey.md" title="Seamonkey-2.53.23">seamonkey-2.53.23</a> includes both a mail client and newsreader in its installation.

<a class="xref" href="../xsoft/thunderbird.md" title="Thunderbird-140.11.0esr">Thunderbird-140.11.0esr</a> is a mail/news client based on the <span class="application">Mozilla</span> code base.

<a class="xref" href="../gnome/evolution.md" title="Evolution-3.60.2">Evolution-3.60.2</a> is a mail client and organizer from the <span class="application">GNOME</span> desktop.
</div>
