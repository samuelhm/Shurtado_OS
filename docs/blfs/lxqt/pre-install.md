## Notes about building LXQt dependencies {#notes-about-building-lxqt-dependencies}

To build LXQt, the following steps are recommended:

<div class="procedure">
1.  First, build the whole <a class="xref" href="../x/installing.md#xorg">Xorg Chapter</a>, including Wayland support with required and recommended dependencies. Test that the environment works with twm as the window manager. With experience, some of the packages can be omitted, but that is not recommended for first time users.

2.  Second, build the required parts of Qt. The full package, <a class="xref" href="../x/qt6.md" title="Qt-6.11.1">Qt-6.11.1</a> is quite long and has a lot of dependencies.

3.  Finally build the LXQt desktop using the pages below in order. The packages in the applications section are optional, but as a minimum, <a class="xref" href="qterminal.md" title="qterminal-2.4.0">qterminal-2.4.0</a> is recommended.
</div>
