# x264-20250815 {#x264-20250815}

## Introduction to x264 {#introduction-to-x264}

<span class="application">x264</span> package provides a library for encoding video streams into the H.264/MPEG-4 AVC format.

<div class="installation" lang="en">
## Installation of x264 {#installation-of-x264}

Install <span class="application">x264</span> by running the following commands:

```bash
./configure --prefix=/usr   \
            --enable-shared \
            --disable-cli   &&
make
```

This package does not come with a test suite.

Now, as the <code class="systemitem">root</code> user:

```bash
make install
```
</div>

<div class="commands" lang="en">
## Command Explanations {#command-explanations}

*`--disable-cli`*: This switch disables building the command-line encoder which is redundant since it requires FFmpeg for most of the input formats.

<code class="option">--disable-asm</code>: Use this switch if you didn't install NASM.
</div>
