# mpg123-1.33.5 {#mpg123-1.33.5}

<div class="sect1" lang="en">
<div class="package" lang="en">
## Introduction to mpg123 {#introduction-to-mpg123}

The <span class="application">mpg123</span> package contains a console-based MP3 player. It claims to be the fastest MP3 decoder for Unix.

<div class="installation" lang="en">
## Installation of mpg123 {#installation-of-mpg123}

Install <span class="application">mpg123</span> by running the following commands:

```bash
./configure --prefix=/usr &&
make
```

To test the results, issue: <span class="command"><strong>make check</strong></span>.

Now, as the <code class="systemitem">root</code> user:

```bash
make install
```
</div>
