# Window Managers {#window-managers}

<div class="chapter" lang="en">
Window Managers and Desktop Environments are the primary user interfaces into the <span class="application">X</span> Window System. A window manager is a program that controls the appearance of windows and provides the means by which the user can interact with them. A Desktop Environment provides a more complete interface to the operating system, and provides a range of integrated utilities and applications.

There are many Window Managers available. Some of the more well known ones include <span class="application">fvwm2</span>, <span class="application">Window Maker</span>, <span class="application">AfterStep</span>, <span class="application">Enlightenment</span>, <span class="application">Sawfish</span>, and <span class="application">Blackbox</span>.

The Desktop Environments available for Linux are <span class="application">GNOME</span>, <span class="application">KDE</span>, and <span class="application">XFce</span>.

Choosing a Window Manager or Desktop Environment is highly subjective. The choice depends on the look and feel of the packages, the resources (RAM, disk space) required, and the utilities included. One web site that provides a very good summary of what is available, screenshots, and their respective features is <a class="ulink" href="https://xteddy.org/xwinman/index.md">Window Managers for X</a>.

In this chapter, the installation instructions of several Window Managers and one lightweight Desktop Environment are presented. Later in the book, both <span class="application">KDE</span> and <span class="application">GNOME</span> have their own sections.
