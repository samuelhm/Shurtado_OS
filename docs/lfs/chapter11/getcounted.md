# []{#ch-finish-getcounted}11.2. Get Counted {#get-counted .sect1}

::: {.sect1 lang="en"}
Now that you have finished the book, do you want to be counted as an LFS user? Head over to [https://www.linuxfromscratch.org/cgi-bin/lfscounter.php](https://www.linuxfromscratch.org/cgi-bin/lfscounter.php){.ulink} and register as an LFS user by entering your name and the first LFS version you have used.

Let\'s reboot into LFS now.
