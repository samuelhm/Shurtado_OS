# []{#ch-intro-changelog}1.3. Changelog {#changelog .sect1}

:::: {.sect1 lang="en"}
This is version r13.0-131-systemd of the Linux From Scratch book, dated June 2nd, 2026. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via [https://www.linuxfromscratch.org/mirrors.html](https://www.linuxfromscratch.org/mirrors.html){.ulink}.
