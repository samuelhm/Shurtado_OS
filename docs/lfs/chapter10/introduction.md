# []{#ch-bootable-introduction}10.1. Introduction {#introduction .sect1}

::: {.sect1 lang="en"}
It is time to make the LFS system bootable. This chapter discusses creating the `/etc/fstab`{.filename} file, building a kernel for the new LFS system, and installing the GRUB boot loader so that the LFS system can be selected for booting at startup.
