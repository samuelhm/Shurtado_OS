# 06 — Qt 6

> Fase 5 del plan. 4 paquetes. ~14 SBU total.
> Qt6 es el mas pesado de esta fase: 13 SBU, 47 GB de espacio.

## Paquetes

### 5.1 Qt-6.11.1

- **Para que sirve:** Framework de aplicaciones multiplataforma. Widget toolkit para GUI y base de todo KDE (Frameworks, Plasma, apps KDE).
- **Prefijo:** `/opt/qt6` (no en `/usr`).
- **Flags de build:**
  - `-prefix /opt/qt6` — instalacion aislada versionable
  - `-release` — build optimizado
  - `-dbus-linked -openssl-linked` — enlace explicito a D-Bus y OpenSSL
  - `-system-sqlite` — usar SQLite del sistema
  - `-nomake examples -no-rpath -no-sbom -journald`
  - Se eliminan: `qt3d`, `qtquick3dphysics`, `qtwebengine`, `qtopcua` (no usados)
- **Que instala:**
  - **Programas:** `qmake6`, `assistant`, `designer`, `linguist`, `qdbus`, `qml`, `qmllint`, `qtdiag`, `qtpaths`, `qdoc`, `lupdate`, `lrelease`, `qsb`, `qt-cmake`, y ~35 mas.
  - **Librerias:** `libQt6Core.so`, `libQt6Gui.so`, `libQt6Widgets.so`, `libQt6Network.so`, `libQt6Qml.so`, `libQt6Quick.so`, `libQt6OpenGL.so`, `libQt6DBus.so`, `libQt6Sql.so`, `libQt6Svg.so`, `libQt6WaylandClient.so`, `libQt6PrintSupport.so`, `libQt6Multimedia.so`, `libQt6Bluetooth.so`, y ~60+ mas.
- **Dependencias:**
  - **Obligatorio:** Xorg Libraries.
  - **Recomendados (tenemos/instalamos):** alsa-lib, make-ca, double-conversion, GLib, FFmpeg, harfbuzz, ICU, jasper, libjpeg-turbo, libpng, libtiff, libwebp, libxkbcommon, Mesa, mtdev, Wayland, libinput.
- **Post-instalacion:** Agregar `/opt/qt6/lib` a `ld.so.conf`, crear `/etc/profile.d/qt6.sh`, configurar `sudoers.d/qt`.

### 5.2 qca-2.3.10

- **Para que sirve:** Qt Cryptography Architecture — API de criptografia multiplataforma con plugins. Requerido por KDE Frameworks (KAuth, KWallet).
- **Que instala:** `libqca-qt6.so`, `qcatool-qt6`, `mozcerts-qt6`, y plugins: `libqca-ossl.so`, `libqca-gcrypt.so`, `libqca-gnupg.so`, `libqca-nss.so`, `libqca-softstore.so`.
- **Dependencias:** CMake, Qt6, make-ca, Which.

### 5.3 qcoro-0.13.0

- **Para que sirve:** C++20 coroutines para Qt. Permite usar `co_await` con tipos Qt. Requerido por Plasma.
- **Que instala:** `libQCoro6Core.so`, `libQCoro6DBus.so`, `libQCoro6Network.so`, `libQCoro6Qml.so`, `libQCoro6Quick.so`, `libQCoro6WebSockets.so`.
- **Dependencias:** Qt6 (unica dependencia).

### 5.4 qtkeychain-0.16.0

- **Para que sirve:** Almacenamiento seguro de contraseñas usando el keychain del sistema (org.freedesktop.secrets). Plasma lo usa para KWallet y wifi passwords.
- **Que instala:** `libqt6keychain.so`.
- **Dependencias:** Qt6.
