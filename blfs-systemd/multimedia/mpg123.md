## Introduction to mpg123 {#introduction-to-mpg123 .sect2}

The [mpg123]{.application} package contains a console-based MP3 player. It claims to be the fastest MP3 decoder for Unix.

## Installation of mpg123 {#installation-of-mpg123 .sect2}

Install [mpg123]{.application} by running the following commands:

``` userinput
./configure --prefix=/usr &&
make
```

To test the results, issue: [**make check**]{.command}.

Now, as the `root`{.systemitem} user:

``` root
make install
```

### Short Descriptions

  ------------------------------------------------------------ ---------------------------------------------------------------------------
  []{#mpg123-prog}[[**mpg123**]{.command}]{.term}              is used for playing MP3 files via the console
  []{#mpg123-id3dump}[[**mpg123-id3dump**]{.command}]{.term}   is a tool to dump ID3 meta data from MPEG audio files using libmpg123
  []{#mpg123-strip}[[**mpg123-strip**]{.command}]{.term}       extracts only MPEG frames from a stream using libmpg123 (stdin to stdout)
  []{#out123}[[**out123**]{.command}]{.term}                   plays raw PCM audio to an output device
  []{#libmpg123}[`libmpg123.so`{.filename}]{.term}             contains the [mpg123]{.application} API functions
  []{#libout123}[`libout123.so`{.filename}]{.term}             contains the [out123]{.application} API functions
  []{#libsyn123}[`libsyn123.so`{.filename}]{.term}             contains some audio signal synthesis and format conversion functions
