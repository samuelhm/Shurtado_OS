## Introduction to x264 {#introduction-to-x264 .sect2}

## Installation of x264 {#installation-of-x264 .sect2}

Install [x264]{.application} by running the following commands:

``` userinput
./configure --prefix=/usr   \
            --enable-shared \
            --disable-cli   &&
make
```

This package does not come with a test suite.

Now, as the `root`{.systemitem} user:

``` root
make install
```
:::

::: {.commands lang="en"}
## Command Explanations {#command-explanations .sect2}

*`--disable-cli`*: This switch disables building the command-line encoder which is redundant since it requires FFmpeg for most of the input formats.

`--disable-asm`{.option}: Use this switch if you didn't install NASM.
:::
