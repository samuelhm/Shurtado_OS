# []{#server-mail}Mail Server Software {#mail-server-software .chapter}

:::: {.chapter lang="en"}
MTAs are the programs which transport mail from one machine to the other. The traditional MTA is [Sendmail]{.application}, however there are several other choices. BLFS does not include the [Sendmail]{.application} package anymore, but both [Exim]{.application} and [Postfix]{.application} provide the same functionality as well as a compatibility symlink for applications that expect [Sendmail]{.application} to be present.

In addition to the SMTP servers, there is also a POP3/IMAP server (Dovecot).
