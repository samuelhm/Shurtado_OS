# Mail Server Software {#mail-server-software}

<div class="chapter" lang="en">
MTAs are the programs which transport mail from one machine to the other. The traditional MTA is <span class="application">Sendmail</span>, however there are several other choices. BLFS does not include the <span class="application">Sendmail</span> package anymore, but both <span class="application">Exim</span> and <span class="application">Postfix</span> provide the same functionality as well as a compatibility symlink for applications that expect <span class="application">Sendmail</span> to be present.

In addition to the SMTP servers, there is also a POP3/IMAP server (Dovecot).
