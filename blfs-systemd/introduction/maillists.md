# []{#maillists}Mailing Lists {#mailing-lists .sect1}

::: {.sect1 lang="en"}
The linuxfromscratch.org server is hosting a number of mailing lists that are used for the development of the BLFS book. These lists include, among others, the main development and support lists.

For more information regarding which lists are available, how to subscribe to them, archive locations, etc., visit [https://www.linuxfromscratch.org/mail.html](https://www.linuxfromscratch.org/mail.md){.ulink}.
:::
